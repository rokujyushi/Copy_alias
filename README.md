# CopyAlias

AviUtl ExEdit2 用の `aux2` プラグインです。エイリアス文字列（`.object` 相当）を扱う複数の補助機能を提供します。

## 主な機能

- オブジェクトメニュー: **エイリアスをコピー**
	- 選択中オブジェクトのエイリアスを `CRLF` 連結して `CF_UNICODETEXT` でクリップボードにコピー
- オブジェクトメニュー: **エイリアスの値をペースト**
	- クリップボード内 INI から項目を選択し、現在選択中オブジェクトへ部分適用
- レイヤーメニュー: **クリップボードからオブジェクト貼り付け**
	- クリップボード内のエイリアスをオブジェクトとして生成（複数対応）
- オブジェクトメニュー: **エイリアスからパスをコピー**
	- エイリアス内のパスらしき値を抽出し、選択してクリップボードへコピー

## 仕様

- 複数選択時: 全件のエイリアスを改行（CRLF）連結してコピー
- 選択0件時: 何もしない
- フォーカスオブジェクトへのフォールバック: なし
- フォーマットは単一オブジェクトのエイリアス（`.object`）と同様
```ini
[Object]
frame=87,167
[Object.0]
effect.name=図形
図形の種類=円
サイズ=100
縦横比=0.00
ライン幅=4000
色=ffffff
角を丸くする=0
[Object.1]
effect.name=標準描画
X=0.00
Y=0.00
Z=0.00
Group=1
中心X=0.00
中心Y=0.00
中心Z=0.00
X軸回転=0.00
Y軸回転=0.00
Z軸回転=0.00
拡大率=100.000
縦横比=0.000
透明度=0.00
合成モード=通常

```

## 使い方

1. レイヤー編集でオブジェクトを選択
2. 右クリックメニューから「エイリアスをコピー」を実行
3. クリップボードに `CF_UNICODETEXT` として格納される
4. 右クリックメニューから「エイリアスの値をペースト」を実行
5. ダイアログのツリーで反映したいプロパティにチェック
6. 「適用」を押すと、現在選択中の全オブジェクトへ反映

## INI選択適用

1. クリップボードにINI形式テキスト（`.object` 相当）を用意
2. 右クリックメニュー「エイリアスの値をペースト」を実行
3. ダイアログのツリーで反映したいプロパティにチェック
4. 「適用」を押すと、現在選択中の全オブジェクトへ反映

### INI表示/反映ルール

- `[Object]` セクションのメタ情報（`frame` など）は表示しない
- `effect.name` が無い `[Object.N]` セクションは表示しない
- 反映対象は「現在選択中の全オブジェクト」
- エフェクト不一致・項目不一致はスキップして継続
- 同名エフェクトが複数ある場合は出現順（0始まり）で対応付け
- 反映はINIの順番で行う（後勝ち）

## クリップボードからオブジェクト貼り付け

- クリップボードのテキストを複数エイリアスに分割して順に作成
- 配置基準:
	- 選択オブジェクトがある場合: 先頭選択オブジェクトの `layer` / 開始フレームを基準
	- 選択オブジェクトがない場合: 現在カーソル位置を基準
- エイリアス側の `layer` / `frame` の相対差を保って貼り付け

## エイリアスからパスをコピー

- 選択中オブジェクトのエイリアスを解析し、パス候補を重複除去して一覧表示
- `C:\...` / `\\server\share\...` などの Windows パス形式を対象
- 候補を選択するとクリップボードへコピー

## 言語ファイル

- 同梱: `i18n/English.copyalias.aul2`
- `aviutl2.toml` で `Language/English.copyalias.aul2` へ配置するよう定義

## ビルド

- 前提: Rust（`cargo`）
- デバッグビルド: `cargo build`
- リリースビルド: `cargo build --release`
- 生成DLL:
	- `target/debug/CopyAlias.dll`
	- `target/release/CopyAlias.dll`
- `aviutl2.toml` のアーティファクト設定で `Plugin/CopyAlias.aux2` として配置可能

> 補足: `cpp/` 配下には旧実装（C++）が残っていますが、現行は Rust 実装（`src/`）です。
