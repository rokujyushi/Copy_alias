# CopyAlias

AviUtl ExEdit2 用の `aux2` プラグインです。選択中オブジェクトのエイリアスをクリップボードにコピーします。

## 仕様

- 複数選択時: 全件のエイリアスを改行（CRLF）連結してコピー
- 選択0件時: 何もしない
- フォーカスオブジェクトへのフォールバック: なし
- フォーマットは単一オブジェクトのエイリアス(.object)ファイルと同様
```ini
[Object]
frame=87,167
[Object.0]
effect.name=図形
図形の種類=円
サイズ=100
縦横比=0.00
ライン幅=4000
色=ffffff
角を丸くする=0
[Object.1]
effect.name=標準描画
X=0.00
Y=0.00
Z=0.00
Group=1
中心X=0.00
中心Y=0.00
中心Z=0.00
X軸回転=0.00
Y軸回転=0.00
Z軸回転=0.00
拡大率=100.000
縦横比=0.000
透明度=0.00
合成モード=通常

```

## 使い方

1. レイヤー編集でオブジェクトを選択
2. 右クリックメニューから「エイリアスをコピー」を実行
3. クリップボードに `CF_UNICODETEXT` として格納される

## ビルド

- `x64 Release`
- `build_release.ps1` を実行
- 出力: `bin/x64/Release/CopyAlias.aux2`
